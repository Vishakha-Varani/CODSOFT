## 🎯 Tasks Completed

### 🎬 Task 1: Movie Genre Classification

This project builds a Machine Learning model that predicts the genre of a movie based on its plot summary using Natural Language Processing techniques.

---

## 🔹 Objective

To automatically classify movie genres from text descriptions using machine learning.

---

## 🔹 Approach

- Data loading and preprocessing
- Text feature extraction using **TF-IDF Vectorization**
- Model training using **Logistic Regression**
- Model evaluation using accuracy and classification report

---

## 🧠 Machine Learning Workflow

Movie Plot → Text Preprocessing → TF-IDF Vectorization → Logistic Regression Model → Genre Prediction

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn

---

## 📊 Dataset Information

The dataset contains:
- Movie ID  
- Title  
- Genre  
- Plot Description  

It is used to train the model to learn relationships between text and genre labels.

---

## 🎯 Features

- Predict movie genre from text input
- Handles multiple genres such as:
  - Action
  - Comedy
  - Drama
  - Horror
  - Thriller
  - Romance
  - Sci-Fi
